# 循环神经网络

## 对象
- 序列化数据

## 应用
- 机器翻译
- 序列标注
- 图像描述
- 推荐系统
- 智能聊天机器人
- 自动作词作曲

## 求解
- BPTT
- 梯度
   - 梯度爆炸
      - 梯度裁剪
         - 大于阈值时，对梯度等比收缩
   - 梯度消失
      - 长短时记忆模型
         - 输入门
            - sigmoid
         - 遗忘门
            - sigmoid
         - 输出门
            - sigmoid
         - 内部记忆单元
            - tanh
         - 激活函数
            - 使用ReLU将很难实现门控
      - 门控循环单元
   - 如果采用ReLU作为激活函数
      - 假设函数一直处于激活区域
      - t时刻的表达式将包含t个W连乘
      - 结果将趋于0或无穷
      - 只有当W取值在单位矩阵附近时才能取得比较好的效果
      - 可以将W初始化为单位矩阵
